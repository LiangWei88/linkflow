## 输出解释

### 项目定位

**双链大纲笔记软件** - 一款自托管、Web化、支持大纲+双向链接的笔记工具，灵感来源于 Roam Research 和 Logseq。

### 核心功能

1. **大纲编辑器**
   - 支持无限层级嵌套的大纲节点（如 `1 → 1.1 → 1.1.1`）
   - 每个节点可折叠/展开
   - 支持拖拽排序（鼠标/触控）
   - 兼容 Markdown 语法（如 `**粗体**`、`- 列表`）

2. **双向链接**
   - 使用 `[[wikilinks]]` 语法创建链接
   - 支持自动补全：输入 `[[` 时弹出已有块列表
   - 点击链接可跳转到目标块
   - 反链面板：展示当前块的反向链接

3. **Daily Notes**
   - 每日自动创建笔记（如 `2026-03-21`）
   - 支持自定义模板
   - 快捷键（如 `Ctrl+K`）插入 `[[ ]]`

4. **自托管与 Web 化**
   - 支持 Docker 一键部署
   - Web 端访问：浏览器直接使用
   - 离线模式：本地缓存数据

5. **基础搜索与导航**
   - 全局搜索：支持模糊搜索、标签筛选
   - 快速导航：`Ctrl+P` 弹出页面跳转面板

6. **移动端 PWA 适配**
   - 支持安装为 App（iOS/Android）
   - 触控优化：大纲编辑器、双向链接操作适配触控

### 技术栈

| 组件 | 技术选型 | 备注 |
|------|----------|------|
| 前端 | React + Tiptap | 纯 UI 层，无需 API 调用 |
| 后端 | Node.js（Express） | 仅用于静态文件服务 |
| 数据库 | SQLite | 前端直接访问，无需 API 中转 |
| 部署 | Docker | 一键部署，数据持久化 |
| 搜索 | SQLite FTS5 | 内嵌全文索引 |

### 分层设计

| 层次 | 责任 | 实现方式 |
|------|------|----------|
| **UI 层** | 界面渲染、交互 | React + Tiptap |
| **应用层** | 业务逻辑、数据处理 | TypeScript（共享代码） |
| **数据层** | 数据存储、查询 | SQLite（直接集成） |
| **存储层** | 文件/数据库 | SQLite 文件 + 附件存储 |

## 示例

### 项目简介示例

```markdown
# 双链大纲笔记软件

一款自托管、Web 化、支持大纲+双向链接的笔记工具。

## 核心功能

- **大纲编辑器**：无限层级嵌套，支持折叠/展开和拖拽排序
- **双向链接**：使用 `[[wikilinks]]` 语法，支持反链面板
- **Daily Notes**：每日自动创建笔记，支持自定义模板
- **自托管**：通过 Docker 部署，数据持久化存储

## 技术栈

React + Tiptap + SQLite + TypeScript

## 快速开始

```bash
docker-compose up -d
```

访问 http://localhost:3000 开始使用。
```
